# Bangladeshi TV Channels

This repo is a collections of bangldeshi tv channels for playing on TV with Internet (IPTV) using .m3u8 player.

You can [download .m3u8](https://raw.githubusercontent.com/mahinaistudio/tv-channel/refs/heads/main/iptv.m3u) or use this link directly:
